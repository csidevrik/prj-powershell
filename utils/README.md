# Get-Current-Date PowerShell Script

Este script de PowerShell, `Get-Current-Date`, está diseñado para devolver la fecha y/o hora actual en diferentes formatos personalizados. Utiliza un conjunto de parámetros para determinar qué tipo de información de fecha/hora devolver.

## Parámetros

El script `Get-Current-Date` tiene tres parámetros principales, cada uno de los cuales cambia el formato de la salida de la fecha y hora:

1. **JustDate** (`-JustDate`): Devuelve solo la fecha en un formato específico.
2. **JustHour** (`-JustHour`): Devuelve solo la hora en un formato específico.
3. **DayAndHour** (`-DayAndHour`): Devuelve tanto la fecha como la hora en un formato específico.

Si no se especifica ninguno de estos parámetros, el script devolverá una fecha completa por defecto.

## Mapa de Meses

El script utiliza un hash table para mapear los números de mes a sus abreviaturas en español:

```powershell
$months = @{
    1  = "ENE"
    2  = "FEB"
    3  = "MAR"
    4  = "ABR"
    5  = "MAY"
    6  = "JUN"
    7  = "JUL"
    8  = "AGO"
    9  = "SEP"
    10 = "OCT"
    11 = "NOV"
    12 = "DIC"
}
```
